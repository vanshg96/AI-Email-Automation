# AI Email Automation

AI Email Automation is a Python-based automation tool that generates and sends emails automatically using AI. The system helps users create professional emails quickly and automate the email sending process.

## 🚀 Features

* AI-generated email content
* Automated email sending
* Simple and easy-to-use Python scripts
* Supports SMTP email sending
* Useful for marketing, notifications, and communication automation

## 🧠 Technologies Used

* Python
* OpenAI API
* SMTP (Simple Mail Transfer Protocol)
* Automation Scripts

## 📁 Project Structure

```
AI-Email-Automation
│
├── email_generator.py
├── send_email.py
├── requirements.txt
└── README.md
```

## ⚙️ Installation

1. Clone the repository

```
git clone https://github.com/vanshg96/AI-Email-Automation.git
```

2. Navigate to the project directory

```
cd AI-Email-Automation
```

3. Install dependencies

```
pip install -r requirements.txt
```

## ▶️ Run the Project

Generate an email using AI:

```
python email_generator.py
```

Send an automated email:

```
python send_email.py
```
